# helth_data_analysis
## Overview
A machine learning model that predicts cardiovascular disease based on health indicators.

## Tools
Python, Pandas, NumPy, Scikit-learn, Seaborn

## What I did
- Cleaned dataset and handled missing values
- Converted age and created BMI feature
- Removed outliers
- Performed exploratory data analysis
- Trained Logistic Regression model with tuning

## Result
Accuracy: ~72%

## Conclusion
Health indicators like BMI and blood pressure strongly influence disease risk.
